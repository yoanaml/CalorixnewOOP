# 🥗 Calorix – Fitness & Nutrition Management System

Calorix is a C++ console application for tracking nutrition, exercise, fitness goals, and user management with admin and trainee roles.
<img width="527" height="360" alt="image" src="https://github.com/user-attachments/assets/a6d99e39-93ae-4868-98cb-aefe90722c99" />

## Features

- 👤 User system (Admin / Trainee)
- 🔐 Login & registration
- 🍗 Food logging and calorie tracking
- 🏋️ Exercise tracking
- 📊 Daily summary & progress reports
- ⚖️ BMI & BMR calculations
- 🎯 Fitness goals (weight loss, bulking, maintenance)
- ⭐ Favorite exercises
- 📁 File save/load system
